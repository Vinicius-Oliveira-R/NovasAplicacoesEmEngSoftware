Descrição do Projeto

Sistema de Gerenciamento de Campeonatos e Assistente Médico Web

Este projeto é uma aplicação Django que integra o gerenciamento de campeonatos esportivos com um sistema de assistência médica. O sistema permite:

  • Gerenciamento de Campeonatos: Criar, editar e gerenciar campeonatos com múltiplas modalidades
  • Controle de Jogadores: Registrar jogadores, vincular a campus e gerenciar suas participações
  • Inscrições: Gerenciar inscrições de times em campeonatos com confirmação
  • Jogos e Resultados: Registrar jogos, resultados e determinar vencedores
  • Sistema Médico: Controlar consultas médicas, atendimentos e especialidades
  • Integração Cross-Domain: Proxy models para reutilizar entidades entre domínios

Desenvolvido como projeto final do 1º trimestre da disciplina de Novas Aplicações em Engenharia de Software
